# Project name: SHRINE

**SHRINE** is the public name for this codebase (formerly *Aegis*).

## Full name

**Simulation of Hydrology, Reservoirs, and Integrated Network Engine**

Note: the last word is **Engine** (singular), not “Engines.”

## Usage

| Context | Form |
|---------|------|
| Branding, titles, README | **SHRINE** |
| Python package / imports | `shrine` (lowercase) |
| Legacy metadata base class | `ShrineObject` in `global_attributes/shrine_object.py` |

## Repository folder

The GitHub repository may still be named `Aegis` until it is renamed on GitHub. The Python package directory is **`shrine/`**.
